## 冒泡排序
时间复杂度O(n^2)
空间复杂度1
```python
def bubble_sort(arr):
    def swap(i, j):
        arr[i], arr[j] = arr[j], arr[i]
    n = len(arr)
    x = -1
    swapped = True
    while swapped:
        x = x +1
        swapped = False
        for i in range(1, n - x):
            if arr[i-1] > arr[i]:
                swapped = True
                swap(i-1, i)
    return arr
```
